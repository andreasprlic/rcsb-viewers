When the model or other aspects of the scene or viewer changes, update notifications need to be forwarded to the concerned parties. The implementation is an 'observer' pattern, where interested parties register with the update controller, and are notified by it when necessary.

The primary class is {@linkplain org.rcsb.mbt.controllers.update.UpdateController}. In addition there are two support classes:

{@linkplain org.rcsb.mbt.controllers.update.UpdateEvent}
An update event contains information for the update. It is created by the update controller and then forwarded to the registered recipients.
{@linkplain org.rcsb.mbt.controllers.update.IUpdateListener}
Registrants must implement this interface to receive updates.
