This is the model description package. Salient classes here are:

    {@linkplain org.rcsb.mbt.model.StructureComponent}
    Base class for all of the structure components that make up a structure — atoms, bonds, helices, coils, etc.
    There is a class derived from StructureComponent for each of the structure component types.
    {@linkplain org.rcsb.mbt.model.StructureComponentRegistry}
    Contains enumerations for each of the structure component types.
    Note: These are currently strings per a JDK 1.0 implementation - expect them to change to enums, in the future.
    {@linkplain org.rcsb.mbt.model.Structure}

    Class that serves as a container for all of the components. This is pretty much a dumb container - it doesn't know about relations of the things it contains. That's reserved for the 'StructureMap' class (see next.)
    {@linkplain org.rcsb.mbt.model.StructureMap}

    This is a partner class contained by the 'Structure' class that describes all of the relationships of the contained components. This also contains the sub-scene information for viewers.
    {@linkplain org.rcsb.mbt.model.StructureModel}
    Consists fundamentally of a list of structures. It also has mechanisms to trigger notifications when a structure changes.
